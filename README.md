# VueSneakers
Учебный проект для изучения возможностей фреймворка Vue.js и Tailwind CSS

## Используемые технологии:
- Vue.js
- Tailwind CSS

## Запуск проекта

```sh
npm install
```

### Компиляция и Hot-Reload для разработки

```sh
npm run dev
```

### Компиляция для продакшена

```sh
npm run build
```
